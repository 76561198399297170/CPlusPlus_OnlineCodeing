# 题目编写规范文档

## 概述

本文档旨在规范题目编写格式，确保题目内容能够被前端正确解析和展示。



## 题目结构

一个完整的题目包含以下文件：

**question.txt**

| 分隔符       |
| ------------ |
| ### 输入格式 |
| ### 输出格式 |
| ### 样例     |
| ### 问题描述 |

**config.json**

| 键           | 值       | 类型  |
| ------------ | -------- | ----- |
| difficulty   | 难度     | Int   |
| memory_limit | 内存限制 | Array |
| tags         | 标签     | Array |
| time_limit   | 时间限制 | Array |



## 编写规范

### 文本格式

- 使用Markdown格式编写题目内容
- 对于需要显示的尖括号，使用`<` 和 `>` 即可
- 对于需要解析的尖括号，使用 `\\<` 和 `\\>`
- 支持使用 `~~删除线~~` 表示删除线
- 样例格式使用以下格式：

````
样例1
```
\\is输入数据\\ie
\\-
\\os输出数据\\oe
```
样例2
```
\\is输入数据\\ie
\\-
\\os输出数据\oe
```
````
其中：

`\\is` 和 `\\ie` 标记输入内容的开始和结束

`\\os` 和 `\\oe` 标记输出内容的开始和结束

`\\-` 是可选的分隔线



### 难度等级

| 等级 | 描述 |
|------|------|
| 1 | 入门 |
| 2 | 基础 |
| 3 | 中等 |
| 4 | 进阶 |
| 5 | 困难 |



### 标签

标签应使用中文，常见标签包括：
- 数学
- 字符串
- 数组
- 动态规划
- 贪心
- 搜索
- 图论
- 数据结构
- 模拟
- 枚举



## 前端解析方法说明

### 文本解析

前端使用以下步骤解析题目文本：

1. **HTML实体转义**：
   - 将 `<` 转换为 `&lt;`
   - 将 `>` 转换为 `&gt;`
   - `\<` 和 `\>` 原样

2. **样例解析**：
   - 识别 `\is`、`\ie`、`\os`、`\oe` 标记
   - 将输入和输出分别提取并显示

3. **格式解析**：
   - 解析Markdown格式（标题、加粗等）
   - 解析删除线 `~~内容~~`



## 示例题目

**question.txt**

````tex
### 问题描述
　　给定一个以秒为单位的时间t，要求用 “<H>:<M>:<S>”的格式来表示这个时间。<H>表示时间，<M>表示分钟， 而<S>表示秒，它们都是整数且没有前导的“0”。例如，若t=0，则应输出是“0:0:0”；若t=3661，则输出“1:1:1”。
### 输入格式
　　输入只有一行，是一个整数t（0<=t<=86399）。
### 输出格式
　　输出只有一行，是以“<H>:<M>:<S>”的格式所表示的时间，不包括引号。
### 样例
样例1
```
\\is0\\ie
\\-
\\os0:0:0\\oe
```

样例2
```
\\is5436\\ie
\\-
\\os1:30:36\\oe
```

````

## 常见问题

### 样例格式错误

**问题**：样例不显示或显示异常
**解决方法**：确保样例格式正确，使用 `\\is`、`\\ie`、`\\os`、`\\oe` 标记

### 删除线不显示

**问题**：删除线效果不显示
**解决方法**：使用 `~~删除线~~` 格式
